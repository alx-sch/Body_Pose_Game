# Body Pose Game
How to play:

Download folder 'model' and  play_7.py

Run play_7.py and pass arguments: 1) Seconds for each pose, 2) number of rounds; e.g. playing 10 rounds a 3 seconds each: pyhton play_7.py 3 10
